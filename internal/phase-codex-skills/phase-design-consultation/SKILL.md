# phase-design-consultation

Use this skill to create a design system from zero.

## Goal

Create a coherent visual system with both safe choices and deliberate risks.

## Workflow

1. Study the category and current norms.
2. Propose safe options and strategic departures.
3. Help choose which risks to take.
4. Capture the chosen rules so later implementation follows them.

## Outputs

- design direction options
- category-informed recommendations
- conservative versus distinctive choices
- design source-of-truth artifact

## Guardrails

- design should be intentional, not generic
- category-breaking choices must still cohere as a system

## Exit condition

After producing the outputs for this skill:

- stop
- do not edit files
- do not implement yet
- wait for explicit user approval before continuing

If the same user message explicitly asks for design consultation plus implementation, deliver the consultation output first and only then continue.
